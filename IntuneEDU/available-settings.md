# <a name="available-settings-for-intune-for-education"></a>교육에 Intune에 대 한 사용 가능한 설정

설정의 구성은 Windows 10 장치를 보호 하 고 올바른 종류의 학습 환경을 향상 시키기 위해 액세스를 만들 수는 주요 방법 중 하나입니다. 교육에 Intune에 개별 그룹 및 해당 하위 그룹에 모두 이러한 환경을 사용자 지정할 수 있도록 하는 많은 설정을 제공 합니다.

대부분의 설정은 하거나 수 있도록 설정 되어 **블록** 또는 **허용** 특정 장치 기능에 대 한 액세스를 **구성 되어 있지** 장치의 기본 설정 또는 사용자가 선택한 설정 됩니다.

> [!NOTE]
> 이러한 사용자, 앱 및 장치 설정을 다 [테 넌 트 설정](what-are-tenants.md)합니다.

먼저 선택 하 여 이러한 설정을 찾을 수는 **그룹**, 선택 **설정을**합니다.

  ![그룹에 대 한 설정 페이지](./media/settings-001-list-of-settings.png)

에 있는 정보의 많은 **역할** 교육 콘솔에 대 한 Intune에서 열을 찾을;이 형식은 쉽게 특정 항목에 대 한 설정을 검색할 수 있도록 설계 되었습니다.

## <a name="basic-device-settings"></a>기본 장치 설정

|설정|기능|
|---|---|
|카메라|장치 카메라에 대 한 사용자 액세스를 차단 합니다.|
|이동식 저장소|USB 드라이브 및 외부 하드 드라이브와 같은 이동식 저장소를 사용 하 여 사용자를 차단 합니다.|
|날짜 및 시간|사용자가 장치 변경 날짜 및 시간 설정 합니다.|
|장치 이름|이름을 변경 하는 장치에서 사용자를 차단 합니다.|
|언어 설정|사용자가 장치 언어를 변경 합니다.|
|장치를 등록 취소|수동으로 관리에서 장치 등록을 취소 하지 못하게 차단 합니다.|
|진단 데이터 보내기|수집 하 여 Windows 개선을 돕기 위해 Microsoft로 익명 사용 데이터를 보낼 것인지 여부를 정의 합니다.|
|프로비저닝 패키지 추가|사용자가 추가 장치 설정을 포함 하는 새 프로비저닝 패키지를 추가 합니다.|
|프로비저닝 패키지 제거|사용자가 장치 설정이 포함 된 프로비저닝 패키지 제거를 차단 합니다.|
|위치 서비스|위치 서비스를 사용 하 여 장치의 위치에 액세스 하는 앱을 차단 합니다.|
|인터넷 공유|사용자 장치의 인터넷 연결을 공유할 때 인터넷 공유를 사용 하지 못하도록 차단 합니다.|
|Cortana|Cortana에 질문에 답변 하 고 작업을 수행할 수 있는 Windows 10에 기본 제공 되는 디지털 도우미를 차단 합니다.|
|장치 영역 설정|국가 및 언어와 같은 지역 설정을 변경할 사용자를 차단 합니다.|
|로컬 하드 드라이브에 파일 저장|사용자가 파일을 로컬로 저장을 차단 합니다.|
|절전 모드 및 전원 설정|전원 변경할 사용자를 차단 하 고 설정을 절전 모드.|
|시작 메뉴 크기|시작 메뉴를 전체 화면 표시 강제 전환할지 여부를 정의 합니다.|
|Windows 추천|이러한 장치에서 모든 Windows 스포트라이트 기능을 차단 합니다.|
|OneDrive 동기화|이러한 장치 및 사용자에 대 한 블록 OneDrive 동기화 합니다.|

## <a name="microsoft-edge-settings"></a>Microsoft Edge 설정

|설정|기능|
|---|---|
|팝업|새 창을 열 웹 사이트를 차단 합니다.|
|Do Not 추적 헤더|Microsoft Edge 웹 사이트 사용자 데이터를 추적 하지 달라고 요청 해야 합니다.|
|쿠키|쿠키는 웹 사이트 설정을 저장할 수 있습니다 또는 트랙 사용자의 동작을 검색 합니다.|
|양식 항목을 자동으로 채우기|블록 온라인 양식 필드에 입력 된 데이터를 저장 합니다.|
|암호 관리자|암호 관리자를 사용 하 여 암호를 저장 하도록 사용자를 차단 합니다.|
|개발자 도구|사용자가 개발자 도구에 액세스 하지 못하도록 차단 합니다.|
|브라우저 확장|확장을 사용 하 여 Microsoft 및 기타 소스에서 추가 기능을 가장자리를 사용자 지정할 때 사용자가 차단 합니다.|
|InPrivate 브라우징 기능|데이터 검색 기록 및 쿠키와 같은 저장 가장자리를 중지 한 InPrivate 브라우징 사용 하 여 사용자를 차단 합니다.|
|정보에 대 한 액세스: 플래그 페이지|에 대 한 액세스를 차단 된에 대 한: 실험적 설정 및 기능을 포함 하는 플래그 페이지입니다.|
|SmartScreen 재정|사용자가 SmartScreen 필터에 의해 차단 된 웹 사이트에 대 한 경고를 무시 합니다.|
|파일에 대 한 SmartScreen 재정|사용자가 SmartScreen 필터의 확인 되지 않은 파일을 다운로드 하는 방법에 대 한 경고를 무시 합니다.|
|엄격한 유해 정보 차단 필터를 사용 하 여 블록 성인 등급 콘텐츠|"블록"으로 설정 성인 등급 콘텐츠를 차단 하는 엄격한 보다는 보통 유해 정보 차단 필터를 사용 합니다.|
|검색 제안|URL 또는 검색 용어를 입력할 때 가능한 웹 사이트를 제안할 블록 가장자리입니다.|
|인트라넷 트래픽을 Internet Explorer에서|"블록"으로 설정 하는 경우 내부 트래픽이 Edge 대신 Internet Explorer에 전송 됩니다.|
|JavaScript|가장자리에서 실행 중인에서 JavaScript를 차단 합니다.|
|기본 검색 엔진|Bing 이나 Google, Yahoo Microsoft Edge에 대 한 기본 검색 엔진으로 선택 합니다. 사용자 또는 다른 관리자가 Intune 통해 전체 경험에서 사용자 지정 검색 엔진을 설정 하는 경우 여기에 기본적으로 해당 사용자 지정 검색 엔진을 정의할 수 있습니다.|
|Microsoft Edge 홈 페이지를 구성 합니다.|홈 페이지 찾아보기 기능이 Microsoft Edge 새 세션을 시작 누군가가 때마다 열을 선택 합니다.|
|Microsoft Edge 즐겨찾기 구성|Microsoft Edge 즐겨찾기 목록에 표시 되도록 웹 사이트를 선택 합니다.|
|블록 즐겨찾기 편집|Edge 브라우저의 즐겨찾기를 편집 하 여 사용자를 차단 합니다.|

## <a name="device-update-settings"></a>장치 업데이트 설정

|설정|기능|
|---|---|
|분기 준비 상태 수준|비즈니스에 대 한 Windows 업데이트에 대 한 장치는 현재 분기 또는 현재 분기에 있는지 여부를 정의 합니다.|
|업데이트 및 유지 관리 기간|업데이트 및 업데이트 설치에 대 한 유지 관리 기간을 정의 합니다.|
|예약 된 설치 날짜|장치를 업데이트 하 고 다시 시작 해야 합니다는 요일을 정의 합니다.|
|예약 된 설치 시간|장치 업데이트 되 고 다시 시작 되는 시간을 정의 합니다.|
|외부의 업데이트 예약 된 시간|전원에 연결 되어 (예: 장바구니)에서 예약 된 업데이트 시간은 외부에서 업데이트 해야 하는 장치를 사용 합니다.|
|기능 업데이트를 연기|장치를 사용할 수 있는 한 후 기능 업데이트를 적용 하도록 대기 하는 일 수를 정의 합니다.|
|품질 업데이트를 연기|사용할 수 있는 후 장치에 품질 업데이트를 적용 하려면 며칠을 정의 합니다.|
|사용자가 가져오기 Insider Preview 빌드에 참여 하도록 허용|사용자가 선택할 수 있는지 여부의 장치 다운로드 및 미리 보기 소프트웨어 설치에 사용할 수 있는 정의 합니다.|
|시험판 기능|날씨 정의 설정, 설정 및 실험에 대 한 시험판 기능 또는 시험판 기능이 없는 사용자가 볼 수 있습니다.|
|배달 최적화|업데이트가 장치에 배달 되는 방법을 정의 합니다.|

## <a name="windows-defender-settings"></a>Windows Defender 설정

> [!NOTE]
> 특정 Windows Defender 설정을에서 사용할 수 있습니다는 [테 넌 트](what-are-tenants.md) 수준입니다.

|설정|기능|
|---|---|
|Windows Defender 설정에 대 한 사용자 액세스 차단|사용자는 장치에서 Windows Defender 설정을 수정할 수 없도록 차단 합니다.|
|실시간 모니터링|항상 맬웨어, 스파이웨어 및 기타 위협에 대 한 검색을 사용 합니다.|
|동작 모니터링|특정 한 알려진된 패턴의 의심 스러운 활동에 대 한 확인 하는 Defender를 사용 하도록 설정 합니다.|
|네트워크 검사 서비스|검색 하 고 악의적인 트래픽을 차단 Microsoft Endpoint Protection Center에서 알려진된 취약점의 서명을 사용 하 여 네트워크 기반의 익스플로잇 으로부터 장치를 보호할 수 있습니다.|
|모든 다운로드 한 파일 검색|맬웨어에 대 한 모든 다운로드 한 파일을 자동으로 검색 합니다.|
|Microsoft 웹 브라우저에서 실행할 스크립트를 검색 합니다.|웹 사이트에서 Edge 및 Internet Explorer를 실행 하려고 시도 하는 모든 스크립트를 검색 합니다.|
|업데이트 주기|Defender에 대 한 확인 하 고 맬웨어 방지 조기 실행-업데이트를 다운로드 하는 빈도 정의 합니다.|
|파일 및 프로그램 활동 검색|파일 또는 프로그램 열리고 때 사용자의 의심 스러운 활동 경고에 대해 맬웨어를 검색 합니다.|
|격리 된 맬웨어를 제거 하기 전에 (일)|정의 파일을 제거 하기 전에 저장 하는 일 수 (0 = 즉시 삭제) 합니다.|
|클라우드 기반 보호|Defender가 잠재적인 보안 위협에 대 한 Microsoft에 정보를 보낼 때 실시간 보호를 가져옵니다. 이 기능은 자동으로 샘플을 보내기로 "사용자에 샘플 제출 확인 메시지를 표시" 설정 된 가장 잘 작동 합니다.|
|샘플 제출에 대 한 사용자를 게|파일 한 추가 분석을 자동으로 Microsoft로 보낼지 여부를 정의 합니다.|
|검색 하 고 블록을 잠재적으로 원치 않는 응용 프로그램|Defender는 사용자와 장치에 자동으로 설치 하는 소프트웨어를 잠재적으로 원치 블록 알림이 전송 됩니다.|
|이러한 확장명의 파일을 검색 및 실시간 보호에서 제외|사용자가 보안 위협에 대 한 검사 하지 않고 액세스할 수 있도록 하려는 파일 형식을 정의 합니다.|

|시작 메뉴에 사용자 타일|시작 메뉴에 표시 되 고에서 현재 사용자의 정보를 차단 합니다.|
